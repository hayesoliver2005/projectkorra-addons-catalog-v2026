## What this repository is

ProjectKorra Addons Repository is a browser-based HTML catalog that brings together addon entries, side plugins, and related server references in one place. It is intended to centralize ProjectKorra content so Minecraft communities can review available resources without having to jump across multiple pages or sites.

This repo is especially handy for players, server administrators, and maintainers who want a more organized view of bending-related tools and add-ons. It also includes metadata collected from forum posts and GitHub sources, making it easier to compare version information and Minecraft compatibility before deciding what to install or track.
